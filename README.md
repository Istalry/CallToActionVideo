# CTA Video Generator

A web-based tool for streamers to create professional "Call to Action" (CTA) videos for their broadcasts. Export transparent WebM videos directly from your browser.

## Features
- **Zero Install**: Runs entirely in the browser.
- **Transparent Export**: Generates `.webm` files with alpha channel for OBS/Streamlabs.
- **Customizable**: Change text, colors, and upload your channel logo.
- **Animations**: Pre-built, catchy animations to drive engagement.

## Tech Stack
-   React + TypeScript + Vite
-   TailwindCSS
-   HTML5 Canvas API (for rendering)

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```
