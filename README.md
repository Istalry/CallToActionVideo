# CTA Video Generator

> [!CAUTION]
> **Vibe Coded Project**
> This project was "vibe coded" (generated via AI assistance) and will **not be maintained**.

A web-based tool for streamers to create professional "Call to Action" (CTA) videos for their broadcasts. Export transparent WebM videos directly from your browser.

## Features
- **Zero Install**: Runs entirely in the browser (or as a standalone app).
- **Transparent Export**: Generates `.webm` files with alpha channel for OBS/Streamlabs.
- **Customizable**: Change text, colors, and upload your channel logo.
- **Animations**: Pre-built, catchy animations to drive engagement.

## Installation
1.  Download the latest release (installer or zip).
2.  Run `CTA Video Generator-Setup.exe` to install, or extract the zip and run the executable.

## Usage
1.  **Customize**: Use the sidebar to adjust text, colors, and upload your logo.
2.  **Preview**: See changes instantly in the main preview window.
3.  **Export**: Click the "Export Video" button.
    -   Select resolution (up to 4k).
    -   Wait for the rendering process to complete.
    -   The `.webm` file will be saved to your downloads.

## Tech Stack
-   React + TypeScript + Vite
-   TailwindCSS
-   HTML5 Canvas API (for rendering)
-   Electron (for desktop app)
